Interface Graph

A container for a set of function nodes to show relationships.

remarks

A Graph, or NodeGraph, is a type of diagram that shows the relationship of multiple function instances to display how information should be transformed as it is passed through the graph. Each graph can be thought of as a function in itself that takes in a set of input data types, or constant data types, and transforms them in order to produce an output data type. The inputs, constants, and outputs are usually also represented as nodes within this graph.

A graph may not contain circular depdencies between nodes.
